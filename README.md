# QA Testing Project – Pearson+ Platform

## Overview
This project demonstrates end-to-end QA testing of the Pearson+ platform across web, mobile, and desktop environments.

## Scope
- Functional Testing
- UI/UX Testing
- Cross-platform Testing

## Tools Used
- Jira (bug tracking & task management)
- Google Sheets (test cases & checklists)
- Notion (documentation)

## Deliverables
- Test Plan
- Test Cases
- Test Scenarios
- Bug Reports
- Test Summary Report

- Identified and documented 20+ bugs
- Covered major user flows across multiple platforms
